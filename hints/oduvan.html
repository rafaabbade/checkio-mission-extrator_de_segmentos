Como extrair uma sublista de uma lista maior?

Você pode usar uma técnica chamada slicing em Python para extrair uma parte específica de uma lista. O slicing é feito utilizando a notação [inicio:fim], onde inicio é o índice do primeiro elemento que você quer incluir e fim é o índice do primeiro elemento que você quer excluir: 

lista = [10, 20, 30, 40, 50]
sublista = lista[1:4]
print(sublista)  # Saída: [20, 30, 40]
Isso permite que você extraia sublistas facilmente sem precisar iterar manualmente pela lista.

Como incluir o primeiro ou o último elemento em um slicing?

Para incluir o primeiro elemento, você pode começar o slicing do índice 0. Para incluir o último elemento, você pode usar índices que ultrapassam o tamanho da lista ou deixar o espaço para o índice vazio: 

lista = [10, 20, 30, 40, 50]
sublista_inicial = lista[0:3]  # Inclui o primeiro elemento
print(sublista_inicial)  # Saída: [10, 20, 30]

sublista_final = lista[2:]  # Inclui o último elemento
print(sublista_final)  # Saída: [30, 40, 50]
Isso permite flexibilidade ao criar sublistas que incluem elementos específicos da lista original.
