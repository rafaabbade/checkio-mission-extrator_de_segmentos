Enunciado

Marcos, um estudante de programação, quer extrair sublistas de uma lista maior para análise de dados. Escreva uma função chamada extrator_de_segmentos que receba uma lista e dois índices (início e fim) e retorne a sublista contida entre esses índices, incluindo o início e excluindo o fim, usando slicing.

Entrada: Uma lista de inteiros e dois índices inteiros.

Saída: Uma sublista de inteiros.
